Desarrolla tu Pensamiento Crítico y Computacional

CritComp es una plataforma web interactiva diseñada para estudiantes, que presenta una serie de ejercicios y desafíos que requieren la aplicación conjunta del pensamiento computacional y crítico.

¿Qué ofrecemos?

CritComp te ayuda a desarrollar habilidades esenciales para el siglo XXI combinando el pensamiento computacional con el pensamiento crítico.

Pensamiento Integrado

Combina el poder del pensamiento computacional y crítico para resolver problemas complejos de forma eficiente.

Ejercicios Interactivos

Accede a una amplia variedad de desafíos y ejercicios que ponen a prueba tu capacidad para analizar, diseñar y evaluar soluciones.

Recursos para Educadores

Materiales didácticos, guías y herramientas de evaluación para integrar estas habilidades en el aula.

Objetivos de Aprendizaje

CritComp está diseñado para alcanzar los siguientes objetivos educativos:

Resolución Integrada de Problemas

Desarrollar la capacidad de los estudiantes para aplicar el pensamiento computacional y crítico en la resolución de problemas complejos.

Integración de Habilidades

Fomentar la integración de habilidades de pensamiento computacional y crítico en el proceso de aprendizaje.

Reflexión y Aplicación

Promover la reflexión crítica y la aplicación de conceptos de computación en diversos contextos.

Formación Docente

Capacitar a los profesores en la integración de estas habilidades en sus clases.
